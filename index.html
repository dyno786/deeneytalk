📱 How to use
Tips for using on an iPad or phone.

2. Unlock parent mode with your PIN to add or edit buttons.
3. The People tab is PIN-protected so it stays private at school.
4. Scan mode shows one item at a time — press Select This when you see a reaction.
5. Everything saves on this device/browser automatically.
